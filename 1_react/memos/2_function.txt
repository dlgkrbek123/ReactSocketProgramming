CRA가 부트스트래핑을 위해서 괜찮은 선택지여왔지만 요즘엔 vite라는 번들링 툴을 이용을 많이한다.

---------------------------------------------------------------------------------

state
=> UI를 결정하는 데이터
=> 관리를 위해 hooks를 사용

useRef
=> DOM요소나 UI를 직접결정하지 않는 변수를 관리

useState
=> 상태를 정의하고 관리
=> 상태의 변화는 리렌더링을 유발

useEffect
=> 컴포넌트 마운트같은 생명주기나, state의 변화에 대한 사이드 이펙트를 발생
