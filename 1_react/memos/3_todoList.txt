CRA를 통해 투두리스트 구현

> npx create-react-app react-app
> npm run start

---------------------------------------------------------------------------------

ReactStrictMode
=> 컴포넌트의 생명주기를 확인하고 문제있는 부분을 개발자에게 알려줌
=> useEffect가 두번 호출되게 함
  => 해당 컴포넌트는 안전하지 않게 알리는 동작
=> 개발에만 적용되며, 운영에 배포되면 사라짐


import React from "react"

<React.StrictMode>
  <App />
</React.StrictMode>

---------------------------------------------------------------------------------

module.css
=> 전역 스타일이 아닌 module로 해당 파일에만 적용되는 스타일 작성

---------------------------------------------------------------------------------

new Map()으로 Map객체를 생성가능
=> map.keys()로 key의 배열을 접근
=> map.has(key)로 값이 존재하는지 확인 가능
=> map.get()으로 값에 접근
=> map.size로 key의 개수를 알 수 있음

=> new Map(prev).set(key, value)로 immutable하게 관리
